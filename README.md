# Quick Start
```
# 克隆代码到本地
git clone https://github.com/MisakaTAT/GoTop.git
# 重命名文件夹
mv GoTop-master GoTop
# 将文件夹复制到你的Typecho插件目录
cp -R ./GoTop /webroot/usr/plugins/
```
